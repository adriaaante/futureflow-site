# Ассеты сайта

Изображения сейчас подгружаются по CDN-ссылкам Higgsfield (CloudFront).
Они стабильно доступны для вашего воркспейса.

## Как скачать локально (через ваш браузер)

1. Откройте файлы из этой таблицы в браузере и сохраните в эту папку
   с указанным именем.
2. Затем в HTML замените CDN-ссылки на относительные пути
   `/assets/img/<name>.png`.
